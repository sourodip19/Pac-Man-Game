# 🟡 Pac-Man Game

A classic **Pac-Man** game built using **HTML, CSS, and JavaScript**. Navigate through the maze, collect pellets, avoid ghosts, and try to achieve the highest score!

## 🎮 Features

* Classic Pac-Man gameplay
* Smooth player movement using keyboard controls
* Collect pellets to increase your score
* Ghost enemies with basic movement
* Collision detection
* Score tracking
* Game Over and Restart functionality
* Responsive game layout
* Lightweight and runs directly in the browser

## 🛠️ Tech Stack

* **HTML5** – Game structure
* **CSS3** – Styling and animations
* **JavaScript (ES6)** – Game logic, movement, collision detection, scoring, and interactions

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/pac-man-game.git
```

2. Open the project folder

```bash
cd pac-man-game
```

3. Run the game

Simply open `index.html` in your preferred web browser.

No additional installation or dependencies are required.
